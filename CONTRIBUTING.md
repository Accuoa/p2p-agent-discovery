# Contributing

Spec proposals, test vectors, and reference-implementation patches all welcome.

## For spec changes

1. Open an issue describing the change
2. Reference the section of `SPEC.md` you're proposing to change
3. Wait for design discussion before opening a PR

## For test vectors

1. Add the vector under `examples/`
2. Update `reference-impl/` to pass it
3. Open a PR

Sign off your commits.
